При имитационном моделировании необходимо провести экспериментов. В каждом эксперименте моделируется процесс функционирования одного экземпляра системы. Для ой системы моделирование состоит в вычислении значения случайной величины времени работы этой системы до момента отказа. Для каждого периода жизни системы случайная величина вычисляется по своему алгоритму. Моделирование первого периода Для ой системы первоначально надо определить, к какому подмножеству она относится. Для этого используется распределение После того, как номер подмножества определен, необходимо сгенерировать значение как случайной величины, распределенной поэкспоненциальному закону с параметром который определяется по номеру подмножества.
